#### 作业
```
Week02 作业题目：
1. 我们在数据库操作的时候，比如 dao 层中当遇到一个 sql.ErrNoRows 的时候，
是否应该 Wrap 这个 error，抛给上层。为什么，应该怎么做请写出代码？
```
**作答**
```
我认为该情况应该上抛error 到业务层。在业务层有两种处理方式：
1。 如果是通过id 查询tag表详情，没有查询到数据，应该直接在业务层catch ，然后返回给api 有好到错误提示，如没有查找到信息等。
2。 如果通过ID 查询tag表详情， 而该信息需要在service层被其他接口调用到时候，此时也应该在业务层处理该error,并封装有好到异常信息至api层，
```
